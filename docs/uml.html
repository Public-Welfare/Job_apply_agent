Class diagram in UML notation: + public / – private members, and typed relationship arrows. Every job source is a Strategy behind JobSource; a classifier tags each job so the dashboard dropdown can filter across all sources. All boxes are now implemented in the JS version — the ATS + RemoteOK sources are aggregated by a DiscoveryService (fetch → classify → cache), while CrawlerService drives the resume-tailoring agent.
